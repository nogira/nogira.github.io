
# How to Create a Hugo Blog

<p style="color:rgb(200,200,200)">Note: This is for mac.</p>


```shell
# Install Hugo. (Requires homebrew)
brew install hugo
      
# Generate new site. (Site folder 'myblog' is stored in /Users/<User Name>/)
hugo new site myblog
# Change directory to site folder.
cd myblog

# Install theme. (More themes found at https://themes.gohugo.io/)
git init
git submodule add https://github.com/nanxstats/hugo-tanka.git themes/hugo-tanka
# Set theme.
echo theme = \"hugo-tanka\" >> config.toml

# Create first post.
hugo new mypost.md

# Start server to preview site. (The site URL should show up in the terminal)
hugo server -D
# Alternatively, build the static website files
hugo -D
```


<b>Website Customisation</b>

<p>Customisation requires editing of the <code class="language-none">config.toml</code> file. In the case of this theme, you can go to <code class="language-none">/myblog/themes/hugo-tanka/exampleSite/</code>, then copy <code class="language-none">config.toml</code> there and use it to replace the <code class="language-none">config.toml</code> in <code class="language-none">/myblog/</code>.</p>
<p>More information on <code class="language-none">config.toml</code> options can be found [here](https://gohugo.io/getting-started/configuration/).</p>
